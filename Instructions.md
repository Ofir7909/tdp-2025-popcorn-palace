# Instructions

- Clone the repository

```
git clone https://github.com/Ofir7909/tdp-2025-popcorn-palace.git
```

- Install dependencies

```
cd tdp-2025-popcorn-palace
npm install
```

- Start Postgres database on docker

```
docker compose up
```

- Run the server

```
npm run start
```

- To run tests

```
npm run test:e2e
```
